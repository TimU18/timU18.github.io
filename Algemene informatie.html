Algemene informatie


input
Wanneer je op een toets drukt ontvangt de compter ongtvangt informatie. Manieren waarop de computer input kan krijgenzijn bijvoorbeeld: muis, toetsenboord, touchscreen en webcam.
proces
De computer verwerkt de informatie die die heeft gekregen. Proces bestaat uit 3 elementen: Uitvoerbare code(de code moet wel kloppen), Data(de computer heeft informatie nodig om mee te werken.) en uitvoeringscontext.
storage
Na het proces slaat de computer de gekregen informatie op
output
De computer laat het verwerkte informatie wat in de opslag staat zien op het scherm
