Kleurmodellen


Met kleurmoodellen kan je aangeven welke kleur je iets wil geven, de 3 hoofd kleuren zijn: rood, groen en blauw(RGB model). als je een soort code maakt kan je bepalen hoe rood, groen en blauw je iets wil hebben. Dit kan zijn bijvoorbeeld: achtergrond, text of een tabel.

De kleur code die ik zelf gebruik als tekst achtergrond is: (192, 90, 61) en de HEX code is: #42d1f5



Het "CMYK"model



Je hebt niet alleen het "RGB model", maar ook het "CMYK model" dit staat voor: cyaan, magenta, yellow en key (zwart). Dt werkt hetzelfde als met get "RGB model".

Het verschil tussen de 2 modellen is dat Het "RGB mdel" wordt gebruikt voor vooral tv en computerscherm en het "CMYK model" wordt vooral gebruikt voor printen.
