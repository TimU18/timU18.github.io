Kleurmodellen


Met kleurmoodellen kan je aangeven welke kleur je iets wil geven, de 3 hoofd kleuren zijn: rood, groen en blauw(RGB). als je een soort code maakt kan je bepalen hoe rood, groen en blauw je iets wil hebben. Dit kan zijn bijvoorbeeld: achtergrond, text of een tabel.





De kleur code die ik zelf gebruik als text achtergrond is: (192, 90, 61) en de HEX code is: #42d1f5
